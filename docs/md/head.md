# Head

## 1. Name

**head** - output the first part of a file

## 2. Synopsis

**head** \[OPTION\] \[FILE\]

## 3. Description

Print the first 10 lines of FILE to standard output.

## 4. Parameters

**-n, –lines** NUM – print the first NUM lines instead of the first 10
**-v** – display version information and exit
