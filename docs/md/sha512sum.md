# sha512sum

## 1. Name

sha512sum - Print SHA512 message digest of provided files

## 2. Synopsis

**sha512sum** \[FILE\]...

## 3. Description

Print SHA512 message digest of provided FILEs. If no FILEs are provided
then read from stdin.
